Bemvindo ao jogo 'Numeri'.

O objectivo do jogo é desenhar uma única linha fechada que deixe todos os números como válidos. Os números indicam o número de lados da linha que devem ficar adjacentes ao mesmo número.

    

Use '>' para ver a solução, ou para criar um novo puzzle. Use '<' para limpar o puzzle, ou para alternar o modo de desenho (arestas/células).

Pode virar o iPhone para baixo para revelar números omissos grelha.

Procure por Slitherlink na Wikipedia para mais informação.
