Bem-vindo ao jogo 'Numeri'.

O objectivo do jogo é desenhar uma única linha fechada que deixe todos os dígitos válidos. Os dígitos indicam o número de lados da linha que devem ficar adjacentes ao mesmo dígito.

    

Use '>' para ver a solução, ou para criar um novo puzzle. Use '<' para limpar o puzzle, ou para alternar o modo de desenho (linha/células).

Pode virar o iPhone para baixo para revelar os números omissos na grelha.

Para técnicas do jogo procure por Slitherlink na Wikipedia.
