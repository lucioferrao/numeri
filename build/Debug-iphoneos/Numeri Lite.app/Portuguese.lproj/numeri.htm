Bem-vindo ao jogo 'Numeri'.

O objectivo do jogo é desenhar uma única linha fechada que deixe todos os dígitos válidos (brancos). Cada dígito indica o número de segmentos da linha à volta o mesmo dígito.

    

Use '>' para ver a solução, ou para criar um novo puzzle. Use '<' para limpar o puzzle.

Pode virar o iPhone para baixo para revelar os dígitos omissos na grelha.

Para técnicas do jogo procure por Slitherlink na Wikipedia.
